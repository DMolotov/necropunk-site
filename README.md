# Necropunk Site

Небольшой веб‑сайт для поддержки TTRPG‑сессий и быстрой генерации персонажей по системе NecroPunk.

## Возможности
- Статические страницы: лор, правила, карта, существа, оружие.
- Интерфейсы создания персонажей: ручное, случайное. Хранение данных о персонажах пользователя.
- Простая серверная часть на Node.js с примерами маршрутов и middleware.
- Авторизация и доступ онлайн.

## Требования
- Node.js (рекомендуется v14+)
- npm

## Установка
1. Клонируйте репозиторий:
	 `git clone <repo-url>`
2. Перейдите в папку проекта и установите зависимости:
	 `npm install`

## Запуск локально
- В `package.json` есть скрипт `start`:
	`npm start`

## Структура проекта (основное)
- `client/` — фронтенд: HTML и JS (страницы и скрипты).
- `server/` — точка входа для сервера (`server.js`).
- `src/` — вспомогательные скрипты (например, `auth.js`).
- `lib/` — полезные модули (`mongo.js`, `users.js`).
- `middleware/`, `routes/` — middleware и маршруты сервера.
- `data/` — статические данные (например, `routes.json`).
